# Homebrew Tap

Custom Homebrew tap for [dictait](https://github.com/trahloff/dictait).

## Install

```bash
brew tap trahloff/tap
brew install --cask dictait-mac
```

## Available Casks

| Cask | Description |
|------|-------------|
| `dictait-mac` | Global dictation for macOS with AI-powered transcription |
